# Instalação:

`git clone https://github.com/Carlos-Burg/pedra_papel_tesoura---Redis-DB.git`

`cd pedra_papel_tesoura---Redis-DB`


# Configuração do Redis DB

`python -m pip install redis`

`Conectar porta padrão: `
`host: 10.1.69.134`
`port: 6379`
`db: 0`


## Execução:

`python main.py`


# Descrição:

`Este projeto implementa um sistema completo de Pedra-Papel-Tesoura utilizando Redis para armazenar salas, jogadores e jogadas. Isso inclui criação e conexão de salas, Registro de jogadores, Registro de jogadas por rodada, Finalização e Exclusão da sala apenas pelo jogador 1.`


# Principais modificações e melhorias:

`Sistema de criação e exclusão de salas, Controle de jogadores com permissão, Registro de jogadas, Função de finalização.`


# Tecnologias utilizadas:

`Python, Redis DB`
